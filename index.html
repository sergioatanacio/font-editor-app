OpenType Workflow

Font Editor App

Aplicacion web para crear tipografias desde una plantilla SVG, importar contornos, validar readiness y exportar archivos OTF.

Abrir Editor Repositorio

Que hace la app

  • Crea proyectos de fuente con metricas iniciales.
  • Genera plantilla SVG en grilla de glifos.
  • Importa paths y los convierte al espacio tipografico.
  • Valida bloqueos y advertencias antes de exportar.
  • Exporta OpenType con trazas y diagnosticos JSON.

Algoritmo resumido

  • Definir set de caracteres por preset.
  • Mapear coordenadas SVG a unidades em (UPM).
  • Normalizar contornos y validar cierre de paths.
  • Persistir glifo + unicode para construir cmap.
  • Serializar fuente con opentype.js.

Flujo recomendado

  1. InicioProyecto: crea o abre proyecto.
  2. PlantillaSvg: descarga plantilla base.
  3. ImportacionSvg: carga SVG editado y previsualiza.
  4. PrevisualizacionImportacion: confirma aplicacion.
  5. ValidacionExportacion: revisa readiness.
  6. ExportacionTtf: exporta fuente OTF.
